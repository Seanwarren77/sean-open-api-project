# Open API Weather Project

## Overview
This is a simple weather application that uses the [Open-Meto API] (https://open-meto.com/) to display:

1. Current temperature
2. Current weather conditions (wind speed and description)

Users can click on city buttons to view the weather for *Raleigh, NC ** or * New York, NY **
